# 描述模板

## 结构

高播放描述遵循"钩子前置，信任构建，流量沉淀"结构。

1. **首两行**（移动端可见）：必须包含剧情钩子或核心承诺
2. **中间部分**：简要补充剧情或频道定位
3. **结尾固定区域**：标签矩阵、订阅引导、社交媒体链接

## 类型

### 剧情简介型
用1-2句话复述并扩展标题的核心冲突和反转

### 频道欢迎型
开头说明频道内容属性，再接剧情简介或标签

### 合规声明型
开头提示"Viewer discretion advised"，再进入剧情简介

### 多语言导流型
强调"FULL"、"Multi SUB"、"PT DUB"等观看属性

## 规则

- 描述前两行是黄金位置，不要被订阅链接占据
- 剧情简介要克制，讲清核心矛盾即可，不要剧透所有反转
- 订阅和引导语放在剧情简介之后
- 标签按语义分组，不要无序堆砌
