# 封面模板卡（蒸馏版）

> **单一数据源规矩**：本文件从7语种distill.json的hook_cover_mapping + thumbnail_guidelines自动蒸馏生成。
> 数据驱动，无人工翻译偏见。改知识只改distill.json，然后重新生成本文件。
>
> **快照声明**：当前7张卡 = 已验证的hook类型数。不是最终集——当新hook在≥2个语种中被数据验证时，自动准入新卡。低样本卡（emotion 4/7语种、system 5/7语种）的规则密度低于高样本卡，使用时需注入更多few-shot examples。
>
> **pt回退说明**：pt的distill.json无hook_cover_mapping，封面规则走data_driven默认（thumbnail_guidelines）。使用pt封面卡时，cover_pattern取通用规则，不取逐语种pattern。

## 使用流程

```
1. 分析剧情 → 提取hook类型
2. 选卡 → 取cover_pattern + 适用题材 + 匹配骨架
3. 查语种 → 取构图/色彩/符号/文字的本地化规则
4. 注入反推prompt → 从references/cover-reference-prompts.md取同桶2-3条
5. 填变量槽 → {主角} {配角} {场景} {冲突} {色彩}
6. 输出生图prompt → 英文80-120词
```

## 模板卡

### 身份反差局（identity）

**意图**：低位符号与高位符号同框对比
**数据覆盖**：6/7语种
**适用题材**：CEO、豪门、校园、男频逆袭、真假千金、现代都市、古装
**匹配骨架**：#1身份落差打脸型、#8亲情守护打脸型、#9集体误判打脸型
**反推prompt**：见 `references/cover-reference-prompts.md` 的「身份反差局（identity）」章节（3条）

**cover_pattern（封面要表达什么）**：

- **en**：同框展示低位符号与高位符号，如女仆装对冰球巨星、穷人对豪车、西装CEO对清洁工。
- **es**：低位服装和高位场景同框，主角被围观或被轻视，反派表情得意。
- **id**：低位角色与高位符号同框，如穷女孩旁边出现CEO、西装团队、直升机或保镖。
- **jp**：用服装、保镖、豪宅、集团、囚服、医疗道具等视觉证据证明身份差；人物表情采用主角冷静、反派震惊。
- **tr**：主角穿着与其'隐藏身份'不符的卑微服装，但身处奢华环境或被强者人物围绕；或直接展示'揭露瞬间'——CEO单膝跪地、众人目瞪口呆
- **繁中**：低位服裝與高位場景同框，如外賣服對仙裙、賣菜小夥對惡霸、工地小夥對美女總裁。

**title_pattern（标题怎么写）**：

- **en**：低位身份 + unaware/actually/turns out + 高位身份
- **es**：看似贫穷、卑微或普通，结果是继承人、富豪之女、武林高手。
- **id**：低位身份被羞辱，后半句揭示真实高位身份。
- **jp**：低位身份被看不起，后半用「正体」「実は」「誰も知らなかった」揭露真实地位或能力。
- **tr**：卑微身份标签 + 被轻视/羞辱 + 真实身份揭露 + 旁观者震惊
- **繁中**：低位身份 + 殊不知/豈料 + 真實高位身份

**范例（真实竞品）**：

**en**：
- 标题：[Full]My Childhood Crush Became Hockey Star… And Still Wanted Me Under His Roof ❤️‍🔥
- 封面：男主穿红色冰球球衣呈保护和占有姿态，女主穿女仆装依偎，身份差和亲密关系同时可视化。

**es**：
- 标题：Va a escuela con un saco y lo ridiculizan, resulta es heredero rico y reina del campus lo llama amo
- 封面：校园对峙，破旧男孩面对精致校园女王和跟随者，形成阶级与权力反差。

**id**：
- 标题：Dihina dan diusir oleh Keluarganya, Tak disangka Gadis ini dipungut dan dimanjakan CEO Miliarder!
- 封面：落魄女子蜷缩在地，白衣男子戴墨镜站立，背景有直升机和黑西装团队，阶级反差强烈。

**jp**：
- 标题：【日本語吹き替え】魚屋の娘を見下した偽令嬢、母の正体を知って絶句！#中国ドラマ #日本語 #ドラマ
- 封面：左侧弱势校服女孩，右侧傲慢华丽女性，中间强大西装女性被人群簇拥，形成阶层反转视觉流程。

**tr**：
- 标题：Sokağa Atılan Gerçek Mirasçı, Soğuk CEO Tarafından Kurtarıldı; Kimliğini Öğrenen Herkes Şoke Oldu！
- 封面：军装男性在雪地中保护倒地的素衣女性，冷蓝背景+暖黄灯光，中心三角构图，暗示'英雄救美+身份揭露'的双重张力

**繁中**：
- 标题：3米高惡霸攔路打劫賣菜小夥，殊不知他竟是隱世至尊！
- 封面：高大惡霸抱胸壓迫，普通小夥手持蔬菜冷靜站立，周圍混混包圍，形成強烈體型與氣勢對比。

**人物/表情**：

女频以男女主2人为主，突出身份差和情感距离；男频可用1个强主角加奇观，也可用主角与反派对比。人物表情必须明确：委屈、震惊、冷漠、占有、愤怒、后悔。

**本地化规则（构图/色彩/符号/文字）**：

**en**：
- 构图：优先使用2人中近景、中心构图或对角线对抗构图；关系题材用贴近、拥抱、拉扯、凝视，复仇题材用左右对峙或上下压制，男频奇观用主角居中加背景能量/龙/赛车/风暴。
- 色彩：甜宠和暗恋用暖色、粉色、金色；复仇和权力冲突用红黑、蓝金、强冷暖对比；奇幻和男频用高饱和金色能量、闪电、火焰、深蓝背景。
- 符号：CEO用西装、电梯、豪车、办公室；豪门用戒指、礼服、宴会；王室用皇冠、龙纹、宫殿；背叛用第三者、摔蛋糕、婚戒、离婚协议；系统异能用闪电、光效、数据面板、金龙。
- 文字：封面文字不超过5-7个英文单词，提炼核心承诺，如 Secret Billionaire、Reborn for Revenge、CEO's Hidden Wife、Claimed by Superstar。封面文字应补充标题，而不是复制长标题。

**es**：
- 构图：优先使用contrast对峙构图或中心三角构图。高播放封面常把低位主角和高位群体放在左右两侧，形成清晰权力对比；甜宠题材可用中心双人构图，但必须保留关系张力。
- 色彩：mixed色彩表现较好，推荐冷暖对比：冷色背景配暖色人物光，或豪华暖色场景配破旧低位服装。纯暖浪漫封面适合甜宠，但如果没有冲突容易弱。
- 符号：常用符号包括破衣、麻袋、婚纱、西装、豪车、办公室、保镖、跪地、文件、电话、校园、鱼摊、包子摊、系统/荒岛/篝火等。
- 文字：封面文字不宜复述完整标题，只放2-5个西语强词，如“TRAICIÓN”“HEREDERO”“CEO”“RENACÍ”“SE ARRODILLA”。文字服务钩子，不遮挡表情。

**id**：
- 构图：优先使用contrast构图，其次center构图。女频用2-4人关系对峙，男频用主角居中+敌人跪地/包围。避免信息过散的拼贴，除非是背叛/初恋/多角关系题材。
- 色彩：mixed色彩表现较好，建议冷暖对比：蓝/黑代表压迫和神秘，金/红代表财富、权力、愤怒和爱情。暖色适合婚恋冲突，金光适合战神和玄幻。
- 符号：CEO题材用西装、豪车、黑卡、直升机、保镖、豪宅；婚恋题材用离婚协议、玫瑰、床、机场、婚礼；男频用拳风、锁链、血迹、跪地敌人、金龙、棍棒；亲子题材用相似孩子、孕肚、亲子同框。
- 文字：封面文字应短于标题，突出一个核心词：CEO、MILIARDER、CERAI、DEWA PERANG、DIHINA、3 ANAK。文字颜色用红、黄、白高对比，不要把完整标题塞进封面。

**jp**：
- 构图：身份反转、复仇、家庭冲突优先使用contrast构图，如左右对立、前景后景、隔栏分割；甜宠和契约婚可用center构图突出两人亲密；CP投票或多男主选择可用collage三联拼接。
- 色彩：压迫和悬疑用冷灰蓝，溺爱和婚恋用暖金粉，阶层打脸用黑金红，奇幻和萌宝用高饱和光效。高饱和焦点色只给主角或关键道具。
- 符号：婚姻届、戒指、离婚届、牢房铁栏、囚服、听筒、保镖、西装集团、豪宅、轮椅、银针、八卦阵、放大镜、孩子校服、肖像画、闪电、金色光环。
- 文字：封面文字控制在2-6个日文强词，优先使用「正体判明」「全員絶句」「夫が後悔」「実は神医」「偽令嬢崩壊」「溺愛開始」。不要把长标题复制到封面。

**tr**：
- 构图：优先使用中心构图呈现身份揭露、救援、强宠和关键物证；使用左右对比构图呈现离婚、对峙、能力冲突。人物与道具要形成清晰视线链，如脸—枪—结婚证，手部光效—震惊表情，倒地女主—救援男主。
- 色彩：危险虐恋用冷蓝、雨夜、雪地、医院白；豪门身份用暖金、深棕、礼服黑白；男频逆袭用高饱和金色、黄色、闪电蓝。冷暖对撞优于单一柔美色。
- 符号：高价值符号包括枪、结婚证、轮椅、豪宅、家庭合照、病号服、工装、出租车、红圈标记、发光手、魔法光束、围观人群、雨夜伞、雪地倒地。
- 文字：封面文字应少于6个土耳其词，优先写身份或反转词，如【Gerçek Varis】、【CEO Kurtardı】、【Şoke Eden Sır】、【Milyarder Çıktı】；不要把完整标题塞进封面。

**繁中**：
- 构图：對峙構圖是黃金法則：採用'左右對比'或'中心對抗'構圖，兩個人物分別代表衝突的兩極（卑微vs尊貴、受害者vs加害者）。人物之間要有明顯的視覺分界線。景別建議中景到中近景。
- 色彩：整體偏暗調（深藍、灰色、暗金），在主角身上使用高飽和暖光（金、紅、白光特效）製造'聚光燈'效果。冷暖對比是核心——暗調環境+亮光主角=神秘+期待。避免全暖或全冷。
- 符号：核心視覺符號清單：外賣制服（卑微）、鳳冠霞帔（尊貴）、病號服/輪椅（弱勢）、銀針/手術刀（神醫）、戰甲/武器（戰神）、孕檢單/B超（得子）、結婚證/婚紗（婚姻）、跪地（臣服）、發光特效（覺醒/神力）。每封面至少1個強視覺符號。
- 文字：封面文字建議不超過3-5個字，或完全不用文字。如果用文字，應該是最核心的情緒詞（如'逆襲'、'復仇'、'甜寵'），而不是劇情簡介。文字顏色要與背景形成強對比。

**禁区**：

- 题材错位: 标题讲社会逆袭，封面却像奇幻冒险；会扩大受众但也可能造成预期偏差。
- 只美不钩: 男女主漂亮但没有身份符号、冲突动作或关系张力，用户无法判断为什么要点。
- 标题有爆点封面无证据: 标题写 cheating、reborn、secret heir，但封面只有普通合照，点击承诺不可信。
- 信息过散: 封面人物过多、文字过多、符号过多，会让核心钩子在手机端消失。

### 撞破背叛局（relationship）

**意图**：用人物距离表达关系断裂
**数据覆盖**：6/7语种
**适用题材**：爱情、家庭、先婚后爱

**反推prompt**：见 `references/cover-reference-prompts.md` 的「撞破背叛局（relationship）」章节（3条）

**cover_pattern（封面要表达什么）**：

- **en**：用三角关系、距离、背对、冷脸、求抱、摔物等动作表达关系断裂。
- **es**：用公主抱、依偎、半蹲保护、前任跪地等肢体位置表达关系权力变化。
- **id**：用人物距离表达关系：女主孤立，男主靠近第三者，或男主追逐女主，旁边放证据道具。
- **jp**：用三角站位、亲密拥抱、背对离开、隔栏对峙表达关系变化；男女主距离决定甜虐方向。
- **tr**：男女近距离对峙，肢体语言表达'亲密中的冲突'——一人前倾一人后退、一人抓握一人挣脱。雨夜/婚礼现场等高情绪密度场景
- **繁中**：三角關係明確：受害者在前景冷臉或轉身，背叛者與第三者在同一側，身份/證據道具可見。

**title_pattern（标题怎么写）**：

- **en**：丈夫/妻子/初恋/前任 + 背叛或误会 + 主角离开或后悔

- **tr**：亲密关系中的背叛/阴谋 + 发现真相 + 离开/反击
- **繁中**：多年感情/婚姻 + 撞見背叛 + 不哭不鬧反擊

- 构图：优先使用中心构图呈现身份揭露、救援、强宠和关键物证；使用左右对比构图呈现离婚、对峙、能力冲突。人物与道具要形成清晰视线链，如脸—枪—结婚证，手部光效—震惊表情，倒地女主—救援男主。

**繁中**：
- 构图：對峙構圖是黃金法則：採用'左右對比'或'中心對抗'構圖，兩個人物分別代表衝突的兩極（卑微vs尊貴、受害者vs加害者）。人物之間要有明顯的視覺分界線。景別建議中景到中近景。
- 色彩：整體偏暗調（深藍、灰色、暗金），在主角身上使用高飽和暖光（金、紅、白光特效）製造'聚光燈'效果。冷暖對比是核心——暗調環境+亮光主角=神秘+期待。避免全暖或全冷。
- 符号：核心視覺符號清單：外賣制服（卑微）、鳳冠霞帔（尊貴）、病號服/輪椅（弱勢）、銀針/手術刀（神醫）、戰甲/武器（戰神）、孕檢單/B超（得子）、結婚證/婚紗（婚姻）、跪地（臣服）、發光特效（覺醒/神力）。每封面至少1個強視覺符號。
- 文字：封面文字建議不超過3-5個字，或完全不用文字。如果用文字，應該是最核心的情緒詞（如'逆襲'、'復仇'、'甜寵'），而不是劇情簡介。文字顏色要與背景形成強對比。

**禁区**：

- 题材错位: 标题讲社会逆袭，封面却像奇幻冒险；会扩大受众但也可能造成预期偏差。
- 只美不钩: 男女主漂亮但没有身份符号、冲突动作或关系张力，用户无法判断为什么要点。

- 信息过散: 封面人物过多、文字过多、符号过多，会让核心钩子在手机端消失。
