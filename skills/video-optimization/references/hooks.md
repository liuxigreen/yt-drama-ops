# 钩子体系 + 骨架公式 + 反转词库

## 7种钩子（每个标题至少命中2个）

| 钩子类型 | 关键词示例 | 心理机制 |
|---------|-----------|---------| 
| **情绪钩子** | 愤怒/心疼/爽感/震惊/后悔/跪地求饶/Şoke oldu | 情绪启动器 |
| **身份钩子** | CEO/女帝/替身/战神/神医/千金/少爷/billionaire/Dewa Perang | 人设预期 |
| **反转钩子** | 但/岂料/actually/Tak disangka/然而/ところが/其实 | 信息缺口 |
| **时间钩子** | 10年/重生/前世/After 5 Years/38岁/那一夜/重活一世 | 命运叙事 |
| **冲突钩子** | 背叛/羞辱/抛弃/诬陷/出轨/当众/赶出家门/退婚 | 愤怒燃料 |
| **关系钩子** | 丈夫/妻子/前任/初恋/婆婆/老公/wife/esposo | 代入感 |
| **集体钩子** | 所有人/全场/全城/herkes/todos/全員/nobody ousa | 围观心理 |

## 钩子组合示例

| 标题 | 命中钩子 |
|------|---------|
| Useless man mocked like dog, unexpectedly trillion-dollar heir | 身份+冲突+反转+集体 |
| Five years, no ring. I was just a pawn. | 时间+冲突+关系 |
| Drained in past life, reborn she threw away the demand list | 时间+重生+情绪 |

| 5 años de matrimonio, esposo duerme con pasante frente a ella | 时间+关系+冲突 |

## 13种骨架公式（跨语言验证）

### 1. 身份落差打脸型
**结构**：低位身份 + 被误判/羞辱 + 真实高位身份揭露 + 全场震惊
**适用**：CEO豪门、隐藏战神、神医、真假千金
**播放范围**：40000-200000

### 2. 关系背叛补偿型
**结构**：亲密关系中被背叛 + 离开后获得更高价值 + 原伤害者后悔/跪求
**适用**：追妻火葬场、离婚逆袭、前任后悔、替身
**播放范围**：45000-300000

### 3. 重生改命型
**结构**：前世被消耗/杀害 + 重生机会 + 带着记忆主动清算/改写命运
**适用**：重生复仇、大女主、古装宫斗
**播放范围**：40000-120000

### 4. 系统/能力觉醒型
**结构**：事故/受辱后获得系统/超能力 + 能力立刻改变财富/关系格局
**适用**：系统流、男频都市、玄幻、穿越
**播放范围**：20000-60000

### 5. 情绪爆点场景型
**结构**：抓取极端情绪瞬间（婚礼夜/生日/求婚被羞辱）→ 让观众未看剧情先站队
**适用**：虐恋、甜宠、家庭伦理、婚姻危机
**播放范围**：80000-140000

### 6. 隐藏身份揭露型
**结构**：主角身份被刻意隐藏 + 被人看不起 + 身份揭露 + 身份持有者震惊
**适用**：隐藏富豪、真千金、卧底、秘密继承人

### 7. 契约/被迫关系型
**结构**：被迫进入关系（闪婚/契约/替身）→ 假戏真做 → 身份反转
**适用**：契约婚姻、替身新娘、假结婚

### 8. 低位闯高位型
**结构**：底层身份进入高位场所/圈子 → 被嘲笑/排挤 → 亮出真实实力/身份 → 碾压
**适用**：校园逆袭、职场逆袭、豪门闯入

### 9. 全员退缩唯一敢型
**结构**：所有人退缩/不敢/嘲笑 → 弱者完成不可能任务 → 全场震惊
**适用**：战神、英雄、拯救

### 10. 偶遇隐藏身份型
**结构**：偶然相遇（救人/误进/收留）→ 身份隐藏 → 关系升温 → 身份揭露
**适用**：都市奇遇、命运安排

### 11. 复仇清算型
**结构**：被伤害/陷害 → 获得力量/证据 → 让背叛者付出代价（破产/跪求）
**适用**：复仇、打脸、清算

### 12. 甜宠溺爱型
**结构**：女主被冷落/欺负 → 男主（隐藏身份）出手宠爱 → 反派震惊
**适用**：甜宠、宠爱、豪门甜文

### 13. 穿越/重生改命型
**结构**：穿越/重生到过去/异世界 → 带着信息差改写命运 → 让仇人自食其果
**适用**：穿越、重生、异世界
