### 钩子配对规则

**最强配对**：
- `identity + reversal` — 全球最稳定的高频配对
- `emotion + reversal` — 先心疼再翻盘
- `emotion + identity` — 情绪+身份差双锚点
- `compensation + identity` — 身份揭露+碾压回报
- `relationship + reversal` — 关系冲突+意外结果
- `time + compensation` — 重生+复仇闭环

**低效组合**：
- 只有romance没有冲突 — 缺少点击理由
- 只有CEO身份没有误判 — 缺少打脸前提
- 只有复仇结果没有前置伤害 — 缺少愤怒锚点
- 单独使用time/时间钩子 — 变成背景设定而非剧情
- 单独使用compensation — 缺少为什么补偿的理由

**规则**：先用emotion或relationship让观众站队，再用reversal或identity提供点击承诺。标题最多承载2-3个钩子，超过后会变成信息噪音。

## 13种骨架公式（跨语言验证）

### 1. 身份落差打脸型
**结构**：低位身份 + 被误判/羞辱 + 真实高位身份揭露 + 全场震惊
**适用**：CEO豪门、隐藏战神、神医、真假千金
**播放范围**：40000-200000

### 2. 关系背叛补偿型
**结构**：亲密关系中被背叛 + 离开后获得更高价值 + 原伤害者后悔/跪求
**适用**：追妻火葬场、离婚逆袭、前任后悔、替身
**播放范围**：45000-300000

### 3. 重生改命型
**结构**：前世被消耗/杀害 + 重生机会 + 带着记忆主动清算/改写命运
**适用**：重生复仇、大女主、古装宫斗
**播放范围**：40000-120000

### 4. 系统/能力觉醒型
**结构**：事故/受辱后获得系统/超能力 + 能力立刻改变财富/关系格局
**适用**：系统流、男频都市、玄幻、穿越
**播放范围**：20000-60000

### 5. 情绪爆点场景型
**结构**：抓取极端情绪瞬间（婚礼夜/生日/求婚被羞辱）→ 让观众未看剧情先站队
**适用**：虐恋、甜宠、家庭伦理、婚姻危机
**播放范围**：80000-140000

### 6. 被迫关系升温型
**结构**：因替嫁/契约/救命之恩/家族安排进入关系 → 起初不平等或冷淡 → 因真心/能力/身份揭露转为独占宠爱
**适用**：契约婚姻、先婚后爱、替嫁、残疾CEO
**播放范围**：20000-120000

### 7. 隐藏强者救援型
**结构**：高位人物在危机中向看似低微的人求助 → 后者暴露武力/医术/财富完成救援 → 触发婚恋/报恩/权力重排
**适用**：武术、神医、乡村高手、CEO遇险、男频爽剧
**播放范围**：40000-150000

### 8. 亲情守护打脸型
**结构**：家人被欺负 → 隐藏强者因亲情被迫出手 → 完成守护和复仇
**适用**：男频战神、家族复仇、亲情救赎、天才儿童
**播放范围**：80000-210000

### 9. 集体误判打脸型
**结构**：公共场合所有人嘲笑/不敢行动 → 主角做出唯一正确或惊人的行动 → 颠覆全场认知
**适用**：打脸逆袭、救援奇观、隐藏强者、职场
**播放范围**：40000-60000

### 10. 绝嗣/意外得子型
**结构**：高位男性被设定为无后/无法生育 → 低位女性因意外怀孕生子 → 证据曝光后豪门补偿宠爱
**适用**：霸总甜宠、带球跑、豪门认亲、灰姑娘
**播放范围**：60000-120000

### 11. 危险权势反差甜宠型
**结构**：女主误入危险男性的秘密世界 → 开局威胁/误会/契约 → 结局独占/保护/宠爱
**适用**：黑帮恋人、霸总甜宠、超自然爱情
**播放范围**：18000-50000

### 12. 命运道具触发型
**结构**：一个具体物件或小行为触发命运大转折（破碎名表/鸡腿/蛋炒饭/合同） → 道具既是冲突证据也是身份反转入口
**适用**：豪门误会、办公室事故、穿越、隐藏身份
**播放范围**：40000-110000

### 13. 天才儿童破局型
**结构**：儿童/萌宝拥有超常智力/神力/推理力 → 在成人世界的危机中揭穿反派/拯救家人/扭转命运
**适用**：天才儿童、萌宝、心声暴露、奇幻家庭剧
**播放范围**：10000-20000
